# Alexa Supermarket

## Motivação
Este projeto foi criado com o objetivo de facilitar o processo de compras de supermercado para os usuários da Alexa. Com este projeto, os usuários podem adicionar itens à sua lista de compras da Alexa e, em seguida, adicioná-los diretamente ao carrinho de compras do site do Supermercado Beltrame. Isso elimina a necessidade de digitar manualmente cada item na página do carrinho, economizando tempo e esforço para o usuário.

## Resultados
O projeto foi implementado com sucesso e é capaz de ler a lista de compras da Alexa, adicionar os itens ao carrinho do site do Supermercado Beltrame e, em seguida, exibir uma mensagem de confirmação para o usuário. Além disso, o código foi desenvolvido de forma a ser escalável e poder ser facilmente adaptado para suportar outros sites de supermercado.

## Instruções de uso
1. Faça o clone do repositório no seu computador.
2. Execute o arquivo LocalApp.py.
3. Configure sua lista de itens da seguinte maneira:
    - Clique em 'Buscar Produto' e digite o nome do item que deseja adicionar à sua lista de itens.
    - Selecione o seu produto de preferência e configure a quantidade e a palavra chave para ele.
    - No botão 'Minha Lista' abrirá a janela 'Minha Lista de Produtos' onde você pode ver a lista de itens que você adicionou, remover e modificar os itens que você adicionou.
4. Para simular o funcionamento do servidor, basta rodar o arquivo ServerApp.py, que lerá a lista de compras da Alexa e adicionará os itens ao carrinho do site do Supermercado Beltrame.
